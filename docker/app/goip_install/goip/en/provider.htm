Notice:
    In the most case, not nend to input International Tel and Local Recognition. If send problem, you can try to input International Tel and Local Recognition. International Tel means the number will be added to a default telphone number as a prefix. Local Recognition means if a telphone number with Local Recognition at the beginning,it is a local number, will not be added the International Tel. For example, a Provider named P, its International Tel is +86 and Local Recognition is 0.There are 3 receivers A,B,C whoes provider is P.Receiver A's number is 0755********, '0' at the beginning, so the system identified that it is a domestic(local) number, 0755******** sent to the goip; Receiver B's number is +86189********, '+' at the beginning, so the system identified that it is a international number already, +86189******** sent to the goip;Receiver C's number is 189********, two above-mentioned conditions are not satisfied, so add +86 as a prefix to the number , +86189******** sent to goip.
