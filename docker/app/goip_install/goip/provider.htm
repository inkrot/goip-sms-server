  当前位置：修改服务商
提示：
    一般情况下，国家号码和国内号码识别可以不填， GoIP以默认的方式发送短信。 在某些地区，需要填国家号码(如+86)和国内号码识别(如0)，这样GoIP显式地以国际通用短信或国内短信的方式发送短信。服务器短信发送时，首先判断目的号码是否以+号开头，是的话就直接把这个目的号码发送出去；如果不是以+号开头，就把目的号码与国内号码识别进行前缀匹配吗，如果匹配上，就把目的号码用国内短信的形式发送出去；如果匹配不上，就把号码前面加上国家号码，再发送出去。 服务商国际号码是指默认添加的国际号码前缀。服务商国内号码识别指如果一个号码以该值开头，就认为它是一个国内号码，就不再为添加国际号码前缀。例如建立数据服务商电信的国际号码是+86，国内号码识别是0，接收人甲乙丙三个人号码的服务商都是电信；接收人甲的号码为0755********，以0开头，系统识别出这个个国内号码，直接发送0755********给goip；接收人乙的号码为+86189********, 以+开头，系统识别出这已经是一个国际号码，直接发送+86189********给goip;接收人丙的号码为189********，不满足上述2个条件，将在号码前添加+86，再把+86189********发送给goip。
